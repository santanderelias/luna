# Luna - Personal Finance Tracker

Luna is a simple, client-side web application for tracking personal finances. It helps users manage their income and expenses, visualize their spending habits, and project their financial future. All data is stored locally in the user's browser.

## Features

- **Transaction Management:** Add, edit, and delete income and expense transactions.
- **Financial Dashboard:** View key metrics like current balance, monthly spending, and daily spending averages.
- **Reporting & Visualization:** Interactive charts show monthly income vs. expenses, a breakdown of expenses by category, and a projection of your balance over time.
- **Goal Setting:** Set a target cash amount and see projections on when you'll reach it.
- **Recurring Transactions:** Configure recurring income (paychecks) and fixed monthly expenses.
- **Data Persistence:** All data is saved in the browser's `localStorage`, so your information is preserved between sessions.
- **Responsive Design:** A clean, mobile-first interface that works on various devices.
